# ReactNativeFinanceTracker

A React Native mobile application for tracking financial transactions. This project showcases the use of Expo, TypeScript, and modern React Native components to create a seamless and interactive financial tracking experience.

## Features

- Transaction logging with detailed categorization.
- Interactive filter options for transaction types and categories.
- Clean and responsive UI suitable for a variety of screen sizes.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/ReactNativeFinanceTracker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Usage

To start the application:

```sh
expo start
```
